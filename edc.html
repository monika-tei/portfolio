EDC Case for Charlie Tango

Idea: For this project, we've worked on Charlie Tango's case on EDC's new solution targeting users who are selling their homes and want to find potential buyers. The goal was to design and implement a Next.js application that would allow the user to input their desired property details and be presented with mock-data of potential buyers. The user would be able to select the buyers they like and send it to the database.

  • Working with Next.js, CSS modules, NPM
  • Using an API route to post user's details to Supabase database
  • Creating a dashboard page that displays latest requests from Supabase database
  • Building and designing functional forms
  • implementing the existing design system with given icons, font families and colors based on their intent.
