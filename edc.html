EDC Case

Idea: For this project, we've worked on Charlie Tango's case on EDC's new solution targeting users who are selling their homes and want to find potential buyers. The goal was to design and implement a Next.js application that would allow users find potential buyers and submit their details to EDC.

  • Next.js || REACT
  • Using an API route to post user's details to Supabase database
  • Creating a dashboard page that displays latest requests from Supabase database
  • Building and designing functional forms
  • implementing the existing design system with given icons, font families and colors based on their intent.
