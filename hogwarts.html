Hogwarts Student List

Idea: After covering most important JavaScript methods in 5 weeks, this project assignment was about combining all the JS basis into a playful assignment "Hacking Hogwarts Student List". We were given some JSON files with details of students, and had to build an interface that would display it. The "user" is able to filter, sort, select, make prefect and delete a student from the list.

  • Fetching and displaying data from JSON files
  • Data cleaning with methods like string.prototype, array.prototype, object.prototype
  • Option to seach by name
  • Building logic for sorting, filtering and selecting data
  • Ability to "inject" oneself in the array of students
  • Experimenting with some visual assets using ChatGPT
  • Planning and creating diagram for workflow & functionalities
